## 🚀 خطوات الرفع على GitHub ورفع على Render

### الخطوة 1: الحصول على توكن البوت

1. افتح تيليغرام وابحث عن **@BotFather**
2. أرسل له `/newbot`
3. اختر اسم البوت (مثلاً: **بوت الإسلام النور**)
4. اختر username (مثلاً: `Islam_Noor_Bot`)
5. **احفظ التوكن** الذي يعطيك إياه — يبدو هكذا: `1234567890:ABCdef...`

---

### الخطوة 2: رفع الكود على GitHub

1. سجّل دخول على [github.com](https://github.com)
2. اضغط **New repository**
3. سمّه مثلاً: `islam-nour-bot`
4. اجعله **Private** (خاص) للحفاظ على أمان التوكن
5. اضغط **Create repository**

ثم ارفع الملفات:
- اضغط **Add file → Upload files**
- ارفع **كل ملفات مجلد telegram-bot** (بما فيها مجلد data كاملاً)
- مهم: ارفع ملف `ghazi_ajaj.mp4` أيضاً
- اضغط **Commit changes**

---

### الخطوة 3: نشر البوت على Render

1. سجّل دخول على [render.com](https://render.com) — مجاني بالكامل
2. اضغط **New → Background Worker**
3. اربطه بـ GitHub repository الذي أنشأته
4. اختر الـ Branch: `main`

**إعدادات Build & Deploy:**
```
Build Command:   pip install -r requirements.txt
Start Command:   python main.py
```

5. اضغط **Advanced** ثم **Add Environment Variable:**
   - Key: `BOT_TOKEN`
   - Value: **التوكن الذي حصلت عليه من BotFather**

6. اضغط **Create Background Worker**

---

### الخطوة 4: البوت جاهز! 🎉

بعد دقيقتين من Deploy، افتح بوتك في تيليغرام وأرسل له `/start`

---

## 🔄 تحديث البوت

2. Render سيعمل Deploy تلقائياً
